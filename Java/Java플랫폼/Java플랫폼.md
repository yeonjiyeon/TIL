Java 플랫폼 : Java VM + Java API

-Java 프로그램의 개발과 실행을 위한 환경

-사용중인 운영체제에 맞는 Java 플랫폼(소프트웨어)을 설치해야 함 → JDK안에 Java 플랫폼이 포함되어 있다.

-Java VM

- Java 프로그램의 실행 환경을 제공하는 가상 기계
- Java 프로그램의 구동 엔진

- 메로리 정리를 자동으로 수행 → Garbage Collection
